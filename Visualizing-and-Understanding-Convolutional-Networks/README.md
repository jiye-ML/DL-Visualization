### 文件结构
* activations :
    * 获得数据的特征图
    * 输入： 原始图片
    * 输出： 各个层特征图的csv文件
* deconvolution
    * 解卷积响应的层
    * 输入： 各个层特征层csv文件
    * 输出： 解卷积特征图后重构结果
* occlision
    * 对原图遮挡后运行的结果
